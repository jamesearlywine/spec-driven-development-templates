# Task Tracker – App Spec (Overview)

This folder contains the spec-driven artifacts for **Task Tracker**, an application for managing and tracking tasks. Users can work with a task list as part of the core task-management epic.

- **Design**: Application (vision, requirements, personas, events, entities), Security, Observability
- **Features**: Epics and stories (research → spec → plan → tasks). Current epic: **task-management** (story: **task-list**).
- **Application stack**: Root defaults, frontend, backend (services, data stores, cron)
